# SINMOTORS – Luxury Car Dealership Website

SINMOTORS is a website project for a premium car dealership focused on luxury, sports, and SUV vehicles. The site presents an exclusive vehicle offer, company background, team members, available services, and contact details.

The project was created as a student assignment for a technical school in Żuromin. It emphasizes clean design, clear structure, and a professional presentation suitable for a modern automotive brand.

## Features

* Vehicle catalog with categories (Sports, Luxury, SUV)
* Company and team presentation
* Detailed services section
* Contact form with dealership information
* Responsive and modern layout

## Purpose

The goal of this project is to demonstrate front-end development skills and website structure for a real-world business scenario.

## Technologies

* HTML
* CSS
* JavaScript
